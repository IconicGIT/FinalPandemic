## Controls
~~~~~~~~~~~~~~~

Keyboard Player controls:
- UP: W key
- LEFT: A key
- RIGHT: D key
- DOWN: S key
- GUN SHOT: SPACE key
- BOMB: B key

General controls:
- SKIP SCREEN: SPACE key
- INSERT COIN: SHIFT key
- EXIT GAME: ESC key

Gamepad:
- MOVE: JOYSTIC AND D-PAD
- SHOOT: K
- SKIP SCREEN: P
- BOMB: B
- START GAME: START
- EXIT GAME: BACK
